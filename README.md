# Workforce-Information-System

The Workforce Information System is designed to manage employee data and daily office operations in an organized way. Users can register employees, update their profiles, view attendance records, and generate simple reports. The project reduces the chances of data loss and makes information easily accessible whenever needed. It is suitable for small and medium-sized organizations looking for a basic digital management solution.
## Tech Stack

- React 19
- Vite
- TypeScript
- Node.js
- Express.js
- MongoDB
- Mongoose

## Features

- JWT Authentication
- Role Based Access Control
- Employee Management
- Leave Management
- Payroll Management
